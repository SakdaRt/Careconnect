# คู่มือการใช้งานโปรแกรม

## ระบบ CareConnect — แพลตฟอร์มเชื่อมต่อผู้ว่าจ้างกับผู้ดูแลผู้สูงอายุ

**รหัสโครงงาน:** CE68-29  
**ชื่อโครงงาน:** CareConnect

---

## สารบัญ

1. บทนำ
2. ประเภทผู้ใช้งานและสิทธิ์
3. การเข้าสู่ระบบและลงทะเบียน
4. คู่มือสำหรับผู้ว่าจ้าง (Hirer)
5. คู่มือสำหรับผู้ดูแล (Caregiver)
6. คู่มือสำหรับผู้ดูแลระบบ (Admin)
7. ฟีเจอร์ที่ใช้ร่วมกันทุก Role
8. ภาคผนวก

---

## 1. บทนำ

### 1.1 ภาพรวมของระบบ

CareConnect เป็นเว็บแอปพลิเคชันประเภท Two-sided Marketplace สำหรับบริการดูแลผู้สูงอายุในประเทศไทย ระบบเชื่อมต่อ **ผู้ว่าจ้าง (Hirer)** กับ **ผู้ดูแล (Caregiver)** โดยมี **ผู้ดูแลระบบ (Admin)** คอยกำกับดูแล

ระบบเป็น Progressive Web App (PWA) ใช้งานผ่าน Web Browser ได้ทุกอุปกรณ์ รองรับ Responsive Design

### 1.2 ความต้องการขั้นต่ำ

- **Web Browser:** Chrome, Firefox, Safari, Edge เวอร์ชันล่าสุด
- **อินเทอร์เน็ต:** จำเป็นสำหรับทุกฟีเจอร์
- **GPS:** จำเป็นสำหรับ Check-in/Check-out
- **กล้อง:** จำเป็นสำหรับ KYC และถ่ายรูปหลักฐาน

## 2. ประเภทผู้ใช้งานและสิทธิ์

### 2.1 ประเภทผู้ใช้งาน

| ประเภท | ประเภทบัญชี | คำอธิบาย |
|--------|------------|----------|
| **ผู้ว่าจ้าง (Hirer)** | Guest (อีเมล) หรือ Member (เบอร์โทร) | สร้างงาน ว่าจ้าง จ่ายเงิน |
| **ผู้ดูแล (Caregiver)** | Member (เบอร์โทร) เท่านั้น | รับงาน Check-in/out รับเงิน |
| **ผู้ดูแลระบบ (Admin)** | — | จัดการ user, KYC, dispute |

> **หมายเหตุ:** ผู้ที่ต้องการเป็นผู้ดูแลต้องลงทะเบียนด้วย **บัญชีสมาชิก (Member)** ผ่านเบอร์โทรเท่านั้น

### 2.2 ระบบ Trust Level

| ระดับ | ชื่อ | เงื่อนไข | สิทธิ์เพิ่ม |
|-------|------|----------|------------|
| **L0** | เริ่มต้น | ลงทะเบียนสำเร็จ | สร้าง draft, เติมเงิน |
| **L1** | ยืนยันการติดต่อ | ยืนยัน OTP | โพสต์งาน low_risk, รับงาน |
| **L2** | ยืนยันตัวตน | ผ่าน KYC | โพสต์งาน high_risk, ถอนเงิน |
| **L3** | มืออาชีพ | Bank verified + Score ≥ 80 | สิทธิ์เต็ม |

### 2.3 ตารางสิทธิ์

| การกระทำ | Role | L0 | L1 | L2 | L3 |
|----------|------|:--:|:--:|:--:|:--:|
| สมัคร / เข้าสู่ระบบ | ทุก role | ✓ | ✓ | ✓ | ✓ |
| สร้าง draft งาน | Hirer | ✓ | ✓ | ✓ | ✓ |
| เติมเงิน / ดูยอด | ทุก role | ✓ | ✓ | ✓ | ✓ |
| โพสต์งาน low_risk | Hirer | ✗ | ✓ | ✓ | ✓ |
| รับงาน / Check-in/out | Caregiver | ✗ | ✓ | ✓ | ✓ |
| โพสต์งาน high_risk | Hirer | ✗ | ✗ | ✓ | ✓ |
| ถอนเงิน | Caregiver | ✗ | ✗ | ✓ | ✓ |

## 3. การเข้าสู่ระบบและลงทะเบียน

### 3.1 การลงทะเบียน

#### 3.1.1 สมัครด้วย Google OAuth

1. เข้าหน้า **ลงทะเบียน** (`/register`)
2. กดปุ่ม **"สมัคร / เข้าสู่ระบบด้วย Google"**
3. เลือกบัญชี Google
4. หากเป็นผู้ใช้ใหม่ → หน้า **เลือก Role** → หน้า **ยอมรับข้อกำหนด**
5. เข้าสู่ระบบได้ทันที

> [แนบภาพ: หน้า RegisterTypePage — แสดงปุ่ม Google, บัญชีแขก, บัญชีสมาชิก]

#### 3.1.2 สมัครบัญชีแขก (Guest — อีเมล)

เหมาะสำหรับผู้ว่าจ้างที่ต้องการจ้างผู้ดูแลเท่านั้น

1. กดปุ่ม **"สมัครด้วยอีเมล"**
2. กรอก อีเมล, รหัสผ่าน (≥8 ตัว), ยืนยันรหัสผ่าน
3. กดปุ่ม **"สร้างบัญชี"**
4. ระบบส่งรหัส OTP 6 หลักไปยังอีเมล
5. กรอก OTP → กด **"ยืนยันอีเมล"** (OTP มีอายุ 5 นาที, ส่งใหม่ได้ทุก 60 วินาที)
6. ไปหน้า **ยอมรับข้อกำหนด** → เข้าสู่ระบบเป็น Hirer

> [แนบภาพ: หน้า GuestRegisterPage ขั้นตอนที่ 1 — กรอก email/password]

> [แนบภาพ: หน้า GuestRegisterPage ขั้นตอนที่ 2 — กรอก OTP 6 หลัก]

#### 3.1.3 สมัครบัญชีสมาชิก (Member — เบอร์โทร)

เหมาะสำหรับทั้งผู้ว่าจ้างและผู้ดูแล

1. กดปุ่ม **"สมัครด้วยเบอร์โทร"**
2. กรอก เบอร์โทร, รหัสผ่าน, ยืนยันรหัสผ่าน
3. กด **"สร้างบัญชี"** → ได้รับ OTP ทาง SMS
4. กรอก OTP ยืนยัน
5. เลือก Role (ผู้ว่าจ้าง หรือ ผู้ดูแล) → ยอมรับข้อกำหนด → เข้าสู่ระบบ

> [แนบภาพ: หน้า MemberRegisterPage — กรอกเบอร์โทร/password]

> [แนบภาพ: หน้า RoleSelectionPage — เลือก Hirer หรือ Caregiver]

### 3.2 การเข้าสู่ระบบ

หน้า Login (`/login`) มี 3 ทางเลือก:

1. **Google** — กดปุ่ม "เข้าสู่ระบบด้วย Google" → เลือกบัญชี
2. **อีเมล** — กรอกอีเมล + รหัสผ่าน
3. **เบอร์โทร** — กรอกเบอร์โทร + รหัสผ่าน

> [แนบภาพ: หน้า LoginEntryPage — 3 ปุ่มเลือกวิธีเข้าสู่ระบบ]

### 3.3 ลืมรหัสผ่าน

1. กด **"ลืมรหัสผ่าน?"** ที่หน้า Login
2. กรอกอีเมล → กด **"ส่งลิงก์รีเซ็ต"**
3. เปิดลิงก์ในอีเมล → กรอกรหัสผ่านใหม่ → บันทึก

> [แนบภาพ: หน้า ForgotPasswordPage]

> [แนบภาพ: หน้า ResetPasswordPage]

### 3.4 ออกจากระบบ

กดไอคอนโปรไฟล์ที่ TopBar → เลือก **"ออกจากระบบ"**

## 4. คู่มือสำหรับผู้ว่าจ้าง (Hirer)

### 4.1 หน้าหลัก (Hirer Home)

เมื่อเข้าสู่ระบบเป็นผู้ว่าจ้าง จะเข้าสู่หน้า `/hirer/home` แสดง:

- รายการงานทั้งหมดพร้อมสถานะ (รอตอบรับ / กำลังดำเนินการ / เสร็จ / ยกเลิก)
- ปุ่ม **"สร้างงานใหม่"** (+)
- ตัวกรองสถานะงาน
- แบนเนอร์แจ้งเตือนคำขอส่งงานที่รออนุมัติ

> [แนบภาพ: หน้า HirerHomePage — รายการงาน + ปุ่มสร้างงาน + ตัวกรอง]

### 4.2 สร้างงานใหม่ (Create Job — Wizard 4 ขั้นตอน)

#### ขั้นตอนที่ 1: เลือกประเภทงาน

เลือกจาก 6 รูปแบบงาน:

| ประเภท | คำอธิบาย |
|--------|----------|
| พาไปโรงพยาบาล | พาพบแพทย์ รับยา |
| ดูแลผู้ป่วยทั่วไป | อาบน้ำ แต่งตัว ป้อนอาหาร |
| ดูแลหลังผ่าตัด | ดูแลฟื้นฟูหลังผ่าตัด |
| ดูแลผู้ป่วยสมองเสื่อม | เฝ้าระวังพฤติกรรม |
| ดูแลผู้ป่วยติดเตียง | ดูแลขั้นสูง |
| ดูแลอุปกรณ์การแพทย์ที่บ้าน | เครื่องช่วยหายใจ ออกซิเจน |

ระบบจะถามคำถามเพิ่มเติมเพื่อกำหนด Tasks, Skills, Risk Level อัตโนมัติ

> [แนบภาพ: CreateJobPage Step 1 — เลือกประเภทงาน + Dynamic Questions]

#### ขั้นตอนที่ 2: กรอกรายละเอียด

- ชื่องาน, รายละเอียดเพิ่มเติม
- วันเวลาเริ่ม-สิ้นสุด (เลือกจาก dropdown 24 ชม.)
- อัตราค่าจ้างต่อชั่วโมง → ระบบคำนวณค่าจ้างรวมอัตโนมัติ

> [แนบภาพ: CreateJobPage Step 2 — กรอกรายละเอียด วันเวลา ค่าจ้าง]

#### ขั้นตอนที่ 3: สถานที่และผู้ป่วย

- ค้นหาสถานที่ผ่าน Google Places หรือกรอกเอง
- เลือกผู้รับบริการจากรายการที่สร้างไว้ หรือสร้างใหม่

> [แนบภาพ: CreateJobPage Step 3 — ค้นหาสถานที่ + เลือกผู้รับบริการ]

#### ขั้นตอนที่ 4: เลือกผู้ดูแล (ไม่บังคับ)

- แสดง **Favorites** (❤️ สูงสุด 5 คน) + **ผู้ดูแลแนะนำ**
- **เลือกผู้ดูแล → Direct Assignment** (สร้าง + publish ทันที)
- **ไม่เลือก → บันทึก Draft** แล้ว Publish ภายหลัง

> [แนบภาพ: CreateJobPage Step 4 — Favorites + ผู้ดูแลแนะนำ]

### 4.3 Publish งาน

1. จาก Hirer Home กดที่งาน Draft → หน้ารายละเอียดงาน
2. กดปุ่ม **"เผยแพร่งาน"**
3. ระบบ Hold เงิน (ค่าจ้าง + deposit) จากกระเป๋า

**เงินมัดจำ:**

| ค่าจ้างรวม | Deposit |
|-----------|---------|
| ≤ 500 | 100 บาท |
| 501–2,000 | 200 บาท |
| 2,001–5,000 | 500 บาท |
| 5,001–10,000 | 1,000 บาท |
| > 10,000 | 2,000 บาท |

### 4.4 อนุมัติการส่งงาน

เมื่อผู้ดูแลส่งงาน ผู้ว่าจ้างจะได้รับแจ้งเตือน:

1. เข้าดูงาน → เห็น **รูปภาพหลักฐาน** + **บันทึกสรุป**
2. กด **"อนุมัติ"** หรือ **"ปฏิเสธ"** (ต้องระบุเหตุผล)
3. **หากไม่ตอบรับภายใน 1 ชั่วโมง → auto-approve**
4. เมื่ออนุมัติ → จ่ายเงินให้ผู้ดูแล (หักค่าธรรมเนียม 10%)

> [แนบภาพ: JobDetailPage — รูปหลักฐาน + ปุ่มอนุมัติ/ปฏิเสธ]

### 4.5 ยกเลิกงาน

| กรณี | ผลลัพธ์ |
|------|--------|
| ก่อนมีคนรับงาน | คืนเงินทั้งหมด |
| หลังรับงาน ≥24 ชม. | คืนค่าจ้าง + คืน deposit |
| หลังรับงาน <24 ชม. | คืนค่าจ้าง, ริบ 50% deposit |

### 4.6 ค้นหาผู้ดูแล

1. เมนู **"ค้นหา"** (`/hirer/search-caregivers`)
2. กรองตาม ทักษะ, ประสบการณ์, Trust Level, วันว่าง
3. กดดูโปรไฟล์ → **"มอบหมายงาน"** หรือ **❤️ ชื่นชอบ**

> [แนบภาพ: SearchCaregiversPage — ตัวกรอง + รายการผู้ดูแล]

> [แนบภาพ: CaregiverPublicProfilePage — โปรไฟล์ + ปุ่มมอบหมาย]

### 4.7 ผู้ดูแลที่ชื่นชอบ (Favorites)

เมนู **"ชื่นชอบ"** (`/hirer/favorites`) — แสดงรายการผู้ดูแลที่เคยกด ❤️

> [แนบภาพ: FavoritesPage — รายการ Favorites]

### 4.8 จัดการผู้รับบริการ (Care Recipients)

1. เมนู **"ผู้รับบริการ"** (`/hirer/care-recipients`)
2. กด **"เพิ่มผู้รับบริการ"** → กรอกข้อมูล (ชื่อ, ที่อยู่, โรคประจำตัว, อุปกรณ์การแพทย์ ฯลฯ)
3. โปรไฟล์ที่สร้างจะใช้ได้ตอนสร้างงาน

> [แนบภาพ: CareRecipientsPage — รายการผู้รับบริการ]

> [แนบภาพ: CareRecipientFormPage — ฟอร์มกรอกข้อมูล]

### 4.9 กระเป๋าเงิน (Hirer Wallet)

**เติมเงิน:** เมนู **"กระเป๋าเงิน"** → กด **"เติมเงิน"** → ระบุจำนวน → สแกน QR Code ด้วยแอปธนาคาร → ยอดเพิ่มอัตโนมัติ

**หน้าจอแสดง:** ยอดคงเหลือ, ยอด Hold, ประวัติธุรกรรม

**บัญชีธนาคาร:** `/wallet/bank-accounts` → เพิ่มบัญชี (เลือกธนาคาร, เลขบัญชี, ชื่อบัญชี)

> [แนบภาพ: HirerWalletPage — ยอดเงิน + ประวัติ]

> [แนบภาพ: HirerWalletPage — QR Code เติมเงิน]

### 4.10 ประวัติการจ่ายเงิน

1. จาก Wallet กด **"ดูทั้งหมด"** (`/hirer/wallet/history`)
2. แสดงรายการธุรกรรมพร้อมวันที่ จำนวนเงิน ประเภท
3. กดที่รายการ → ดูใบเสร็จ (`/hirer/wallet/receipt/:jobId`)

> [แนบภาพ: HirerPaymentHistoryPage — รายการธุรกรรม]

> [แนบภาพ: JobReceiptPage — ใบเสร็จรายละเอียดงาน]

## 5. คู่มือสำหรับผู้ดูแล (Caregiver)

### 5.1 หน้างานที่เปิดรับ (Job Feed)

เมื่อเข้าสู่ระบบเป็นผู้ดูแล จะเข้าสู่หน้า `/caregiver/jobs/feed` แสดง:

- รายการงานที่ประกาศอยู่ — เฉพาะงานที่ตรง Trust Level
- ตัวกรองตามประเภทงาน วันที่ ช่วงเวลา
- งานที่ทับซ้อนเวลากับงานที่รับแล้วจะถูกกรองออกอัตโนมัติ

> [แนบภาพ: CaregiverJobFeedPage — รายการงาน + ตัวกรอง]

### 5.2 ดูรายละเอียดงานก่อนรับ (Job Preview)

1. กดที่การ์ดงานใน Job Feed → หน้า `/caregiver/jobs/:id/preview`
2. แสดง: ประเภทงาน, Risk Level, วันเวลา, สถานที่, ค่าจ้าง, Tasks, Skills, ข้อมูลผู้ป่วย
3. กด **"รับงาน"** หรือ **"ปฏิเสธ"**

> [แนบภาพ: JobPreviewPage — รายละเอียดงาน + ปุ่มรับงาน]

### 5.3 รับงาน (Accept Job)

1. กด **"รับงาน"** → ระบบตรวจสอบ Trust Level, Score ≥ 40 (high_risk), เอกสาร, เวลาไม่ทับซ้อน
2. หากผ่าน → สร้าง Job Instance + ย้ายเงินจาก Hirer ไป Escrow + สร้างห้องแชทอัตโนมัติ
3. แจ้งเตือนผู้ว่าจ้าง

### 5.4 งานของฉัน (My Jobs)

เมนู **"งานของฉัน"** (`/caregiver/jobs/my-jobs`) แบ่งตามสถานะ:

| สถานะ | ความหมาย |
|-------|---------|
| **Assigned** | รับงานแล้ว รอ Check-in |
| **In Progress** | กำลังทำงาน |
| **Completed** | เสร็จแล้ว |
| **Cancelled** | ถูกยกเลิก |

> [แนบภาพ: CaregiverMyJobsPage — รายการงานแบ่งตามสถานะ]

### 5.5 Check-in (ลงเวลาเริ่มงาน)

1. เข้าดูงานสถานะ Assigned → กดปุ่ม **"Check-in"**
2. ระบบขอสิทธิ์ GPS → บันทึกพิกัดเป็นหลักฐาน
3. สถานะเปลี่ยนเป็น In Progress → แจ้งเตือนผู้ว่าจ้าง

> **สำคัญ:** หากไม่ Check-in ภายใน **30 นาที** หลังเวลาเริ่มงาน → ระบบ **ยกเลิกอัตโนมัติ** (No-Show) คืนเงินเต็มจำนวนให้ผู้ว่าจ้าง + หัก Trust Score **-20**

> [แนบภาพ: CaregiverMyJobsPage — ปุ่ม Check-in + GPS request]

### 5.6 ส่งงานเสร็จ (Check-out)

#### 5.6.1 ส่งงานเสร็จ (ปกติ)

1. จากงาน In Progress กดปุ่ม **"ส่งงานเสร็จ"**
2. ระบบแสดง Modal:
   - **เลือกหัวข้อสรุป** จาก preset
   - **บันทึกเพิ่มเติม** (textarea)
   - **แนบรูปภาพหลักฐาน** (บังคับ) — JPEG, PNG, WebP, HEIC ไม่เกิน 10 MB
3. กด **"ยืนยัน"** → บันทึก GPS + อัปโหลดรูป + ส่งคำขออนุมัติ
4. ผู้ว่าจ้างได้รับแจ้งเตือน — หากไม่ตอบรับใน **1 ชั่วโมง** → **auto-approve**
5. เมื่ออนุมัติ → เงินโอนจาก escrow ให้ผู้ดูแล (หัก 10%)

> [แนบภาพ: CaregiverMyJobsPage — Modal ส่งงาน พร้อมช่องแนบรูป]

#### 5.6.2 ขอจบงานกรณีพิเศษ

2. กรอกเหตุผล + แนบรูปหลักฐาน
3. ส่งคำขอ → **ต้องรอผู้ว่าจ้างอนุมัติเสมอ**

> [แนบภาพ: CaregiverMyJobsPage — ปุ่ม "ส่งงานเสร็จ" กับ "ขอจบงานกรณีพิเศษ" แยกกัน]

## 8. ภาคผนวก

### 8.1 ตารางสถานะงาน (Job Status Flow)

```
draft → posted → assigned → in_progress → completed
                                         → cancelled (ได้ทุกขั้นตอนก่อน completed)
  posted → expired (หมดอายุ)
```

### 8.2 ค่าธรรมเนียมแพลตฟอร์ม

- **Platform Fee:** 10% ของค่าจ้าง (หักจากฝั่งผู้ดูแล)
- **Hirer จ่าย:** total_amount ตามราคาที่ตั้ง (ไม่บวก fee เพิ่ม)
- **Caregiver ได้รับ:** total_amount - platform_fee

### 8.3 กลไก Trust Score

| ปัจจัย | คะแนน |
|--------|-------|
| งานที่ทำเสร็จ | +5 ต่องาน |
| รีวิว 4-5 ดาว | +3 ต่อรีวิว |
| รีวิว 3 ดาว | +1 ต่อรีวิว |
| รีวิว 1-2 ดาว | -5 ต่อรีวิว |
| ยกเลิกงาน | -10 ต่อครั้ง |
| No-Show | -20 ต่อครั้ง |

- **Score = 0** → ถูก ban จากระบบ (ไม่สามารถเข้าสู่ระบบได้)
- **Score < 40** → ไม่สามารถรับงาน high_risk ได้
- คะแนนรวม clamp 0–100 (ไม่มีเพดานรายปัจจัย)

### 8.4 ประเภทงานและ Risk Level

งาน **high_risk** คำนวณอัตโนมัติเมื่อเข้าเงื่อนไข 1 ใน 5 กลุ่ม:

| กลุ่มเงื่อนไข | ตัวอย่าง |
|--------------|---------|
| ประเภทงาน | emergency, post_surgery, dementia_care, medical_monitoring |
| อุปกรณ์การแพทย์ | ventilator, tracheostomy, oxygen, feeding_tube |
| อาการผู้ป่วย | shortness_of_breath, chest_pain, seizure |
| พฤติกรรมเสี่ยง | aggression, wandering + fall_risk |
| งานเฉพาะทาง | tube_feeding, medication_administration, wound_dressing |

- **low_risk** ต้องการ Trust Level **L1+**
- **high_risk** ต้องการ Trust Level **L2+**
